### 13 Autonomous Agents
- **8 Phase agents:** Init → Brainstorm → Requirements → Design → Development → Testing → Security → Deployment
- **2 Support agents:** Scrum Master, Technical Writer
- **3 Compliance agents:** Audit Trail, State Management, Handshake

### 11 Skills/Commands
```
/keel:sprint-planning        - Plan sprints from backlog
```

## 🎯 Quick Start

### One-Command Feature Development
```bash
/keel:implement-feature story="FEAT-001" feature="Your feature description"
```

This automatically invokes:
```
Product Owner → Business Analyst → Solution Architect
         ↓
Software Engineer → QA Engineer → Security Engineer
         ↓
Release Manager
         ↓
Production Ready ✅
```

### Full Development Cycle
```bash
# Step 1: Create requirements
/keel:create-prd goal="User authentication system"

# Step 2: Elaborate specifications
/keel:analyze-story story="FEAT-001"

# Step 3: Full implementation
/keel:implement-feature story="FEAT-001" feature="Email login"

# Step 4: Validate for release
/keel:release-check story="FEAT-001"
```
